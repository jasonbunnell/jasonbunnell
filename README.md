# Jason Bunnell
This is the personal website for [Jason Bunnell](jasonbunnell.com).  The site is made with [JavaSript](https://en.wikipedia.org/wiki/JavaScript) and the [Materialize](https://materializecss.com/) design framework.

Jason was born and raised in Dallas, attended [Baylor University](https://www.baylor.edu/) and now lives in the Finger Lakes region of New York.
